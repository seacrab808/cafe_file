(나중에 다른거 시켰다고 하시는 분들이 많습니다)
퇴식시 음료 맛은 어떠셨는지 여쭤본다. (카페 초기이니 직접 손님한테 피드백 받기)
주문이 밀렸을 경우, 저희 앞쪽에 주문이 많이 밀려있어서 10-15분정도 걸릴 것 같은데 괜찮으신
가요? 라고 꼭 여쭙기.
밑에 대사 그대로 하지 않아도 되지만, 손님이 납득하실 수 있도록 상황에 알맞게 사용한다.
1. 없는 메뉴를 요청했을 경우
(할 수 있는 최대한 도와드린다) 그러시면 저희 메뉴판에 ㅇㅇ 메뉴는 존재하지 않지만, 있는 재
료로 최대한 만들어드리도록 하겠습니다. 없는 메뉴 음료 가격은 0000원 정도로 저희가 받고 있
는데 괜찮으시겠어요? (재차 여쭤본다)
2. 투숙객이 아닌데 1인1음료를 하지 않으신 경우
죄송하지만 손님 저희 매장은 리조트 안에 있는 커피 라운지라, 외부 손님께서는 1인1음료를 받
고 있습니다. 외부 손님들도 많이 찾을 만큼 저희 음료도 맛있고 예쁜 메뉴들이 많은데 한 번 소
개 도와드려도 될까요?
(리조트답게 예쁘게 말하기)
3. 음료 맛에 불평하시는 경우
죄송합니다, 입에 맞지 않으시군요. 정확히 어떤 부분이 불편하셨는지 말씀해주시면 저희가 피드
백 바로 수용해서 더 맛있는 음료로 보답하겠습니다.
(여쭤보고 추가로 소스나 시럽 넣어드리기)
4. 음료 종류가 적다고 뭐라 하시는 경우
저희 메뉴판에는 음료가 많이 없지만 손님께서 요청하실경우 최대한 도와드리고 있습니다. 메뉴
가 적은 대신 음료 하나하나에 정성을 들여서 퀄리티에 집중을 하고 있어서 조금만 양해 부탁드
리겠습니다.
5. 추가적인 것을 요청하시는 경우
(시럽, 우유, 소스, 생크림, 얼음 등 조금만 더 달라고 하시는 경우에는 더 드린다)
(손님에게 잘 보이지 않게 제조해서 맛있게 드시라고 하면서 나간다)
6. 외국인 고객님이 오셨을 경우
(외국인 손님 국적이나 영어/일본어/중국어가 되시는지 여쭤본다.)
(언어에 맞는 외국어 메뉴판을 드리면서 천천히 보시라고 권한다.)
(당황하지 말고 핫/아이스도 묻고 주문 확인까지 똑같이 도와드린다.)
7. 음료가 잘못 나갔을 경우
(죄송한다고 한 후, 포스기에서 확인을 도와드린다고 한다.)
(확인 후, 음료가 잘 못 나갔을 경우 바로 다시 만들어 드린다.)
죄송합니다 손님, 많이 기다리셨을텐데 저희측에서 실수가 있었습니다. 음료는 지금 바로 만들어
드릴테니까 조금만 기다려주시면 감사하겠습니다.
8. 나간 음료가 맞는데 손님께서 착각하신 경우
(포스기 확인을 도와드린 후)
손님, 저희가 포스기 주문서를 확인해봤는데, 이 음료를 주문하신게 맞는 것 같습니다. 혹시 잘못
음료가 나갈까봐 저희가 주문 시에 손님들께 다시 한 번 주문 확인을 도와드리고 있습니다.
(그리고 다시 카드나 방키?로_결제하신 수단_을 다시 확인을 도와드린다)
죄송합니다 손님, 다시 확인해봤는데 이 음료로 주문하신게 맞습니다.
(이렇게 하면 대부분 아 그래요..? 라고 하시고 가신다.)
9. 음료가 늦는다고 뭐라하시는 경우
죄송합니다. 앞쪽에 주문이 많이 밀려 있어서 저희가 최대한 빠르게 순차적으로 도와드리고 있습
니다. 죄송하지만 조금만 기다려주시면 금방 해드리겠습니다.
10. 음료를 쏟았을 경우
괜찮으신가요?? 저희가 치워드리겠습니다.
(기분 나쁜 티 내지 말고 걱정해드리면서 안부부터 여쭙고 치워드린다고 한다.)
11. 터무니 없는 것을 요청하셨을 경우
죄송합니다. 이런 부분은 조금 어려울 수 있는데, 제가 ~~이런 방향으로 최대한 도와드릴 순 있
을 것 같은데 어떤가요??
12. 외부음식 반입, 취식
죄송합니다. 저희 업장에서 외부 음식 반입이 어렵습니다.ㅜㅜ
외부 음식을 가져와서 드시다가 문제가 생기면 저희 쪽에서 책임을 지게 될 수 있는 부분이라 조
금만 양해를 부탁드리겠습니다ㅠㅠ
13. 제가 손님 응대에 있어서 바라는 부분
직원이 카페 메뉴를 정확히 숙지하여서 손님에게 맞는 음료를 추천할 수 있고, 음료를 알기 쉽게
설명할 수 있었으면 좋겠습니다.
리조트 내 카페이다 보니 안 된다고 말하지 말고, 최대한 도와드리고 어려울 경우 정중하게 말씀
을 드려야 할 것 같습니다.
손님과 대화하다 기분 나쁜 경우가 있더라도 참고 예쁘게 말해야 하는 건 어쩔 수 없이 해야한다
고 생각합니다.
이런거 많이 어려워 하시는 분은 조금 힘들지 않을까…
아무래도 제가 어려보이고 여자다보니 말을 함부로 하시는 분들이 많아서 남자분이 들어왔으면
좋겠습니다
